🎨 Custom Placeholders (Slots) Examples

This page demonstrates how to use custom placeholders (slots) to customize Media Chrome components. All examples use the same video source but different slot customizations.

1. Text-based Icons

Replace default SVG icons with simple text using the play and pause slots.
▶ PLAY ⏸ PAUSE 
<media-play-button>
  <span slot="play">▶ PLAY</span>
  <span slot="pause">⏸ PAUSE</span>
</media-play-button>

2. Emoji Icons

Use emojis for a fun, colorful look. Works great for casual or playful interfaces.
▶️ ⏸️ 🔊 🔉 🔈 🔇 ⛶ ⊗ 
<media-play-button>
  <span slot="play">▶️</span>
  <span slot="pause">⏸️</span>
</media-play-button>

<media-mute-button>
  <span slot="high">🔊</span>
  <span slot="medium">🔉</span>
  <span slot="low">🔈</span>
  <span slot="off">🔇</span>
</media-mute-button>

3. Single Animated Icon

Use the icon slot to create a single element that transitions between states. This example changes color and scales based on play/pause state.
⏯ 
<media-play-button>
  <span class="animated-icon" slot="icon">
    <span>⏯</span>
  </span>
</media-play-button>

<style>
.animated-icon {
  transition: transform 0.3s ease, color 0.3s ease;
}
media-play-button[mediapaused] .animated-icon {
  transform: scale(1.1);
  color: #28a745;
}
</style>

4. Custom SVG with Gradients

Create beautiful custom SVG icons with gradients and effects. 
<media-play-button>
  <svg slot="play" viewBox="0 0 24 24">
    <defs>
      <linearGradient id="playGradient">
        <stop offset="0%" style="stop-color:#28a745" />
        <stop offset="100%" style="stop-color:#20c997" />
      </linearGradient>
    </defs>
    <path fill="url(#playGradient)" d="m6 21 15-9L6 3v18Z"/>
  </svg>
  <!-- Similar for pause -->
</media-play-button>

5. Image-based Icons

Use custom image files for icons. Great for branded experiences. 
<media-play-button>
  <img slot="play" src="play-icon.png" alt="Play" />
  <img slot="pause" src="pause-icon.png" alt="Pause" />
</media-play-button>

6. Complete Themed Player

Combine custom slots with CSS variables for a fully branded player experience.
PLAY ▶ PAUSE ⏸ VOL 🔊 VOL 🔉 VOL 🔈 MUTE 🔇 FULL EXIT 
<media-controller class="themed">
  <video slot="media" src="video.mp4"></video>
  <media-control-bar>
    <media-play-button>
      <strong slot="play">PLAY ▶</strong>
      <strong slot="pause">PAUSE ⏸</strong>
    </media-play-button>
    <!-- other controls with custom slots -->
  </media-control-bar>
</media-controller>

<style>
.themed media-play-button {
  --media-control-background: rgba(220, 53, 69, 0.9);
  --media-control-hover-background: rgba(220, 53, 69, 1);
}
</style>

📚 Key Concepts

Understanding Slots

  • Named Slots: Components define slots like slot="play" that you can fill with custom content
  • Hierarchical Slots: Some components offer both general slots (like icon) and specific slots (like play and pause)
  • Light DOM Styling: Your slotted content stays in the page's DOM, so your CSS can style it directly
  • State-based Display: Components automatically show/hide slots based on media state (playing, paused, volume level, etc.)

Available Slots by Component

  • <media-play-button>: icon, play, pause
  • <media-mute-button>: icon, high, medium, low, off
  • <media-fullscreen-button>: icon, enter, exit
  • <media-seek-forward-button>: icon, forward
  • <media-seek-backward-button>: icon, backward

Check each component's documentation for a complete list of available slots.

Tips for Success

  • Always provide accessible alternatives (alt text, aria-labels) when using custom content
  • Test your custom slots in all relevant states (playing, paused, muted, etc.)
  • Keep accessibility in mind - ensure sufficient color contrast and clickable areas
  • Use semantic HTML elements when possible
