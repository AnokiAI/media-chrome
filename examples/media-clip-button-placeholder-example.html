How to Use in Svelte

In a Svelte application, you can use the placeholder slot like this:

<script>
  import '@AnokiAI/media-chrome/extras/media-clip-selector';
  
  let customText = 'Dynamic Content';
</script>

<media-clip-button 
  mediaclipstarttime={10} 
  mediaclipendtime={30}>
  <span slot="placeholder" class="custom-content">
    {customText}
  </span>
</media-clip-button>
